# 一元二次方程

一元二次方程的一般形式是 ax^2 + bx + c = 0，其中 a != 0。
当方程 x^2 - 5x + 6 = 0 可以写成 (x - 2)(x - 3) = 0 时，两个解分别是 x = 2 和 x = 3。
因式分解时先寻找两个数 m 和 n，使得 m + n = b 且 mn = c。

# 三角形相似

三角形相似常用判定包括 AA、SAS、SSS。
AA 表示两个角分别相等，则两个三角形相似。
SAS 表示两组对应边成比例且夹角相等，则两个三角形相似。
SSS 表示三组对应边均成比例，则两个三角形相似。

# 数学知识库构建建议

教培数学资料中经常包含公式、图片和表格。OCR 后应先统一全角半角符号，保留公式上下文，并检查 x^2、sqrt、分式等表达是否被识别错误。
知识切分建议按章节标题、例题、解题步骤分层处理。chunk size 可从 500 到 800 中文字符开始调试，chunk overlap 可设置为 80 到 150 字符。
如果资料是短例题或公式推导，chunk size 可适当减小，避免把多个不相关题目放入同一片段。
